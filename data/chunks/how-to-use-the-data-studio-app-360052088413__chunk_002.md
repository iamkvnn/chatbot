Now go to **File → Embed report**.

![](/hc/article_attachments/48745429525907)

Now you should be on the **Embed Report** screen. Click **Enable embedding**, then make sure it is displaying an Embed Code. Change your **Width** and **Height** to match the resolution of the display you plan to use as a digital sign - we generally recommend 1920x1080 if you're unsure - then click **Copy to Clipboard.**

![](/hc/article_attachments/48745414135955)

Next, go to the OptiSigns portal. Go to **Files/Assets → Apps.**

![](/hc/article_attachments/52550786483731)

Select **Data Studio:**

![](/hc/article_attachments/52550786484627)

Now you can set up your Data Studio app:

![](/hc/article_attachments/52550819880723)

- **Name -** The name of your Data Studio asset. This is for organizational purposes within OptiSigns only and will not display on your screens.
- **Embed Code** **-** The Embed Code you retrieved a moment ago should be pasted into this field.
- **Update Interval -** How often OptiSigns will ping your Embed URL to update the data displayed on it. This is measured in seconds.

|  |

Now, hit **Save**. You'll have created a Data Studio app.

## Deploying a Data Studio App

You can deploy your new Data Studio app as an individual asset, or as part of a [Split Screen](/hc/en-us/articles/360026559573-How-to-Create-and-Use-the-Split-Screen-App).
